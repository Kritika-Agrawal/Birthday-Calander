Open index.html with a browser and follow along.

Expected input : 
1. Json array of objects having "name" and "birthday" properties only.
2. In the birthday property input should be in (month/date/year) format.
3. Year for which you want to find day of given birthdays ( 0=<year<=275759).

Output : 

1. If two persons with same initial have same birthdate(including year) only one will be shown.
2. Two persons with same initials but different birthdate(including year) can be present in a day.
3. Showing intials only of the persons who have already born before the given date.
